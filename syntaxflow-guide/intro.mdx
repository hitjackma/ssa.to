# SyntaxFlow 简介

SyntaxFlow 是 Yaklang 出品的编译器级高级静态分析语言。基于 Yaklang SSA 数据库的专用查询语言，静态代码行为分析的专用 DSL，为代码分析提供强大支持。

<Admonition type="info">
  SyntaxFlow 致力于解决静态分析中的各类难题。通过抹除语言 AST 特性、赋值操作，将分支与循环编译为基本块和 Phi 结构，构建统一的分析模型。
</Admonition>
